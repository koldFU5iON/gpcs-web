  # Game Project Classification Standard

  **A bond-style rating framework for classifying game projects by production capacity and resource backing.**

  ## The Problem

  The video game industry relies on informal labels — **Indie**, **AA**, **AAA** — to classify projects. These terms are inconsistent, conflated, and create unfair competition.

  ## The Solution

  GPCS provides a transparent framework that rates individual **projects** (not studios) using:

  - **Capacity ratings**: AAA/AA/A/BBB/BB/B/C scale
  - **Independence markers**: I0–I3 codes for ownership/control
  - **Source transparency**: Studios, publishers, grants rated independently
